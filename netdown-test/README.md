## Estrutura do Projeto

```
/public/templates/   → Templates dos documentos utilizados
/lib                 → Lógica de geração de documentos e funções de fetch
/app/api            → Rotas da API (Next.js)
/components         → Componentes reutilizáveis da interface
```

## Descrição
Este projeto permite preencher formulários e gerar documentos automaticamente em formato `.docx`, e `.pdf` sem necessidade de autenticação ou base de dados.
